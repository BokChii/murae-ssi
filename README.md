# 무래씨 컵물회 — 랜딩 페이지 프로토타입 v0

## 사용법

1. `index.html` 파일을 더블클릭해서 브라우저로 여세요. (또는 VSCode Live Server 추천)
2. 채팅에서 보내주신 3장의 이미지를 아래 파일명으로 `images/` 폴더에 저장하세요:

| 채팅에 올린 이미지 | 저장할 파일명 | 사용처 |
|---|---|---|
| 1. 영일대 정자 + 바다 | `images/01-yeongildae-ocean.jpg` | Hero 우측, Location 배너 |
| 2. 완성 컵물회 (흰 배경) | `images/02-cup-finished.jpg` | Cup Story 5단계, Goods 메인 |
| 3. 시장에서 컵 든 손 | `images/03-lifestyle-market.jpg` | Lifestyle "산책로" 슬롯 |

3. 새로고침하면 이미지가 자동으로 채워집니다.

## 자산 표기 규칙

페이지에 두 가지 라벨이 보일 거예요:

- 🟢 **초록 ✅ + IMG-XX** — 이미 제공된 이미지가 들어갈 자리 (위 표대로 저장하면 채워짐)
- 🔴 **빨강 # + 번호** — 아직 없는 자산이 들어갈 자리. 번호별 설명은 아래 참고
